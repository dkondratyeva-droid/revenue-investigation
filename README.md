# Revenue Investigation

Scripts and prompts for investigating revenue anomalies.

## Structure
- `alerts/scripts/` — BQ scripts for drop/spike alerts (steps 1-3)
- `alerts/prompts/` — decision prompts for human steps
- `c-level/` — C-level revenue questions (TBD)
- `investigations/` — saved investigation artifacts

## Usage
```
bash alerts/scripts/01_merchant.sh 2026-02-03
```
